# Unity 3D Horror Game

A 3D horror game developed using Unity as part of a multimedia and game development project.

---

## Game Overview

This project focuses on creating an immersive horror experience through:
- environmental storytelling
- lighting design
- suspense mechanics
- interactive gameplay
- atmospheric audio

---

## Features

- 3D horror environment
- Player movement and interaction
- Atmospheric lighting
- Audio effects and ambience
- Horror gameplay mechanics
- Unity physics system

---

## Technologies Used

- Unity
- C#
- 3D Assets
- Lighting & Post Processing
- Audio Design

---
